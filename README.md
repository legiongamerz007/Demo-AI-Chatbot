# 🤖 AI Chatbot – Web Demo

A minimalistic AI chatbot interface built with HTML, CSS, and JavaScript. Ideal for showcasing chatbot UI/UX for AI projects or API integration.

---

## 💡 Features

- Simple and clean design
- Simulated AI responses (mock)
- Typing effect and message history
- Mobile-responsive layout

---

## 📂 Files

- `index.html` – Main UI structure
- `style.css` – Layout & theme
- `script.js` – Logic and UI updates
- `mock-response.js` – Simulated reply logic

---

## 🧠 Why This Project?

This project helps showcase:

- Frontend UI skills
- Integration pattern for AI models
- Your understanding of conversational flow
